Key Features

**Cybersecurity-Focused Chatbot**  
  An interactive console-based chatbot that provides helpful advice on online safety, cybersecurity threats, and best practices.

**Keyword-Based Intelligent Responses**  
  Recognizes common keywords and topics such as:
  - Cybersecurity & online protection
  - Software development
  - Web development
  - Arrays, OOP (Object-Oriented Programming), and debugging
  - General IT-related questions

**Friendly & Engaging Greeting System**  
  - Asks for the user's name at startup  
  - Displays a personalized "Welcome [NAME]" message in large, colorful ASCII art  
  - Plays a custom greeting audio file GreetingRecording.wav on launch

**Eye-Catching Visual Presentation**  
  - Creates and displays large ASCII art at startup  
  - Uses cyan-colored text for better visual appeal  
  - Automatically centers ASCII art based on console window width

**Natural Conversation Flow**  
  - Continuous chat loop until the user types exit 
  - Case-insensitive input handling  
  - Smart fallback responses when the chatbot doesn't understand the query

**Audio Integration**  
  Uses System.Media.SoundPlayer to play a welcome voice recording at the beginning of the session.

**Error Handling & Robustness**  
  - Handling of missing files 
  - Try-catch block to prevent crashes from unexpected errors  
  - Clear user feedback when files are not found

**Educational Focus**  
  Designed to teach and reinforce concepts in:
  - Cybersecurity awareness
  - Basic programming and IT terminology
